# Академия Кибербезопасности

Образовательная платформа по защите от цифровых мошенников.

## О проекте
Интерактивная обучающая платформа с курсами, тренажерами и тестами по кибербезопасности.

## Функциональность
- Курсы по фишингу, телефонному мошенничеству, ПУУ-схемам и др.
- Интерактивные тренажеры
- Тестирование с сертификатами
- Личный кабинет с прогрессом

## Технологии
- HTML, CSS, JavaScript
- Font Awesome
- Локальное хранилище браузера
